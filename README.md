## TailwindCSS + DaisyUI boilerplate ##
#### Using TailwindCSS JIT ####
##### With `autoprefixer` and `cssnano` for production #####

Demo: [https://daisyui-starter.netlify.app/](https://daisyui-starter.netlify.app/)

---

1. Clone the project
2. Install:
```
npm install
```
3. Generate CSS with:
```
npm run dev
```
```
npm run prod
```
